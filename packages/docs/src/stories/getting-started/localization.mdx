# Localization

Widgets support the following languages:

* `de`
* `en` (default - automatically loaded)
* `es`
* `fr`
* `it`
* `ja`
* `pt`

You can request a new language by opening an issue on [GitHub](https://github.com/qumu/widgets/issues/new).

You can override the locale by setting the `lang` attribute on the widget's parent element or on the widget itself.

<Tip color="warning" title="Performance">
  For performance reasons, only the `en` language is loaded by default. To support additional languages, you will need to load them manually.
</Tip>

## Set the lang for the whole page

By setting the lang attribute on the `<html>` tag, the entire page will use the specified language, and all widgets on the page will automatically adopt it.

```html
<html lang="fr">
  <body>
    <!-- Any widget will use the fr language -->
    <div id="widget"></div>
  </body>
</html>
```

## Set the language for multiple widgets

You can set the `lang` attribute on a parent element of specific widgets to override their language.

```html
<html lang="fr">
  <body>
    <!-- This widget will use the fr language -->
    <div id="widget1"></div>

    <!-- Any UI-Kit component inside the div will use the ja language -->
    <div lang="ja">
      <div id="widget2"></div>
      <div id="widget3"></div>
    </div>
  </body>
</html>
```

## Set the language for a specific widget

You can set the `lang` attribute on a specific widget to override their language.

```html
<html lang="fr">
  <body>
    <!-- This widget will use the fr language -->
    <div id="widget1"></div>

    <!-- This widget will use the ja language -->
    <div id="widget2" lang="ja"></div>
  </body>
</html>
```
